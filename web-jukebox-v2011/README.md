# web-jukebox-v1
Ngày xửa ngày xưa (hồi 2011), mình mới biết đến PHP và muốn làm một cái web nghe nhạc cho riêng mình. 
Hồi đó chưa biết dùng Framework gì đâu, cứ PHP thuần mà táng thôi.

## Cài đặt (Hồi đó làm thế này)
1. Cài XAMPP (bản 1.7.7 nhé, bản mới không chạy đâu :v)
2. Copy vào htdocs
3. Import file database.sql vào phpMyAdmin
4. Chạy localhost/jukebox

## Kỹ thuật
- Dàn trang bằng TABLE (vâng, hồi đó mình chưa thạo DIV/Flexbox)
- CSS viết inline hoặc quăng hết vào style.css
- Dùng thư viện jquery-1.6.2.min.js để tạo hiệu ứng tuyết rơi (cho nó lãng mạn)
